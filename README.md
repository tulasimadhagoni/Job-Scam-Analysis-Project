Job Scam & EdTech Complaint Analysis — Real-World Data Analytics Project

Overview:

This project analyzes real scam job alerts and EdTech-related complaints collected from multiple platforms.
I created a structured dataset, cleaned and categorized the data, built dashboards, and extracted insights to identify scam patterns and consumer behavior trends.
This is a practical analytics project designed to showcase:
Data collection,
Data cleaning,
Categorization,
Dashboard creation,
Business insights,
Fraud pattern understanding.

Project Structure:

1_Report/         → Final written report (PDF & DOCX),
2_Dataset/        → Cleaned dataset created from real screenshots,
3_Dashboard/      → Dashboards, charts, and analytics files,
4_Screenshots/    → Original scam alerts & complaint screenshots,
README.md         → Project documentation.

Key Insights Identified:
Most common scam keywords used in fake job alerts,
Platforms where scams are frequently posted,
Patterns in sender behavior (message tone, links, timing),
EdTech customer complaints categorized by issue type,
Fraud triggers and red flags for users,
Visual dashboards highlighting scam trends.
